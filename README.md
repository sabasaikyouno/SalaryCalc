# Usage
### DEMO 1  
"￥" は月給です。  
"労働" はその月の合計労働時間です。  
現在から前後30年分です。
 
### DEMO 2  
"￥" は日給です。  
"労働時間" はその日の労働時間です。  
右下の "￥" は現ページの月給です。  
右下の "労働" は現ページの合計労働鵜時間です。  
12か月分です。  
  
  
"残日数" は有給の残日数です。  
スワイプで横にスライドできます。
